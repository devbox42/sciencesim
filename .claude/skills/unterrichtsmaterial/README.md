# Skill: Unterrichtsmaterial

Erstellt vollständige Materialsets für Einzel- oder Doppelstunden.

## Aufruf

```
/unterrichtsmaterial [FACH] [KLASSE] [THEMA]
```

## Workflow

```
Phase 0-2: Kontext → Planung (Score 10/10)
    ↓
Phase 2b: ARBEITSPFAD (Score 100/100)  ← PFLICHT!
    ↓
Phase 3-4: Materialien → QA
```

## Fach-Workflows

| Fachgruppe | Workflow |
|------------|----------|
| **MINT** (Physik, Chemie, Informatik) | `/templates/mint/WORKFLOW-MINT.md` |
| **Spanisch** | `/templates/spanisch/WORKFLOW-SPANISCH.md` (v3.0) |

## Materialien

### MINT

| Erstellung | Materialien |
|------------|-------------|
| Pflicht | AB, ML, LH, LS, LP, SL |
| Optional | SIM, PPT, QR-*, PLATZKARTEN |

### Spanisch

| Erstellung | Materialien |
|------------|-------------|
| Pflicht | AB, LB, ML, LH, LP + **Arbeitspfad** |
| Optional | PPT, MT, PLATZKARTEN |
