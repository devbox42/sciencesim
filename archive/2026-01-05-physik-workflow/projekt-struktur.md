# Projektstruktur - Lernpfade

> Richtlinie zur Ordnerstruktur nach Fach, Jahrgang und Thema (Rahmenplan MV)

---

## Namenskonventionen

### Ordnernamen
```
fach/klXX/NN-thema-slug/
```

| Element | Format | Beispiel |
|---------|--------|----------|
| Fach | Kleinbuchstaben | `physik`, `chemie`, `informatik` |
| Jahrgang | `klXX` oder `qN` | `kl07`, `kl10`, `q1`, `q2` |
| Nummer | Zweistellig | `01`, `02`, `03` |
| Thema | kebab-case | `dichte`, `kernphysik`, `oop-java` |

### Beispiele
```
physik/kl07/01-dichte/
physik/kl10/05-kernphysik/
chemie/kl09/01-saeuren-basen/
informatik/q2/01-datenstrukturen/
```

## Dateien pro Themenordner

### Pflicht-Dokumente (pro Unterrichtseinheit)
```
NN-thema/
├── lernpfad-*.html          # Digitaler Lernpfad (1-5 Stück)
├── AB-*.tex + .pdf          # Arbeitsblatt (1-2 Stück)
├── ML-*.tex + .pdf          # Musterlösung (1-2 Stück)
├── LH-*.tex + .pdf          # Lehrerhinweise (1-2 Stück)
├── test-*.html              # Online-Test
├── test-*-papier.tex + .pdf # Papier-Test
└── SIM-*.html               # Simulationen (0-5 Stück)
```

### Optionale Dokumente
```
├── spec.md                  # Spezifikation/Planung
├── uebung-*.html            # Zusätzliche Übungen
└── zusammenfassung.pdf      # Handout für SuS
```
