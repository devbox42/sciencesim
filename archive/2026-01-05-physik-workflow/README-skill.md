# Skill: Unterrichtsmaterial

Erstellt vollständige Materialsets für Einzel- oder Doppelstunden.

## Aufruf

```
/unterrichtsmaterial [FACH] [KLASSE] [THEMA]
```

## Materialien (Pflicht)

- LE (Langentwurf) - empfohlen
- AB (Arbeitsblatt) - Pflicht
- ML (Musterlösung) - Pflicht
- LH (Lehrerhinweise) - Pflicht
- LP (Lernpfad) - Pflicht (Nutzung optional)
