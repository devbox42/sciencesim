### 1.2 Kursstruktur: Spätbeginner

**WICHTIG:** "Spätbeginner" ist die informelle Bezeichnung für "in der Einführungsphase neu begonnene Fremdsprachen".

| Aspekt | Wert |
|--------|------|
| **Klassenstufen** | 10, 11, 12 (jahrgangsübergreifend) |

### 1.3 Besonderheit: Jahrgangsübergreifender Unterricht

Der Kurs wird **synchron** unterrichtet:
- Klasse 10, 11 und 12 gemeinsam im Raum
- Wenige Schüler pro Jahrgang (3-4)
- Binnendifferenzierung nach Lernjahr [LJ1], [LJ2], [LJ3]
- Klausuren differenziert nach Jahrgang

### 1.4 Unterrichtstage

- **Montag:** 1 Doppelstunde (90 Min)
- **Dienstag:** 1 Doppelstunde (90 Min)
- **Gesamt:** 4 WStd = 2 Doppelstunden pro Woche

| **Gesamt** | **10 SuS** | |

**Geschlechterverteilung:** überwiegend weiblich (ca. 9:1)

### 2.2 Leistungsverteilung

| Niveau | Anteil | Beschreibung |
|--------|--------|--------------|
| ★★★★ Spitze | ~10% | Kann als Tutor/Expertin eingesetzt werden |
| ★★★ Stark | ~20% | Proaktiv, versteht schnell |
| ★★ Solide | ~40% | Zuverlässig, braucht manchmal Zeit |
| ★ Unterstützung | ~30% | Braucht Scaffolding, Ermutigung |

### 2.3 Differenzierungsgruppen

| Gruppe A (Anfänger) | Gruppe B (Fortgeschritten) |
|---------------------|---------------------------|
| Kl. 10 (alle) | Kl. 12 (alle) |
| Schwächere Kl. 11 | Stärkere Kl. 11 |
| ~5 SuS | ~5 SuS |

**Niveau Gruppe A:** A1-A2 (Fokus: einfache Strukturen, viel Scaffolding)
**Niveau Gruppe B:** A2-B1 (Fokus: komplexere Strukturen, mehr Autonomie)

### 2.4 Didaktische Implikationen

- **Peer-Tutoring:** Stärkere SuS können schwächere unterstützen
- **Gemischte Tandems:** A+B-Paarungen für gegenseitige Unterstützung
- **Scaffolding:** Gestufte Hilfen besonders für Gruppe A
- **Ermutigung:** Unsichere SuS brauchen positive Verstärkung
- **Sprechzeit:** Alle SuS sollen mehrfach pro Stunde sprechen

## TEIL 5: HINTERGRUND LEHRKRAFT

### 5.1 Anerkennungsverfahren (Kontext)

Die Lehrkraft befindet sich im **Anerkennungsverfahren** für die Staatliche Anerkennung als Lehrkraft.

**Relevanz für Materialerstellung:**
- Materialien sollten **didaktisch fundiert** und **rahmenplankonform** sein
- Differenzierung und Binnendifferenzierung sind besonders wichtig
- Unterrichtsentwürfe (LE) sollten dem Landesstandard MV entsprechen

### 5.2 Didaktische Schwerpunkte

Bei der Materialerstellung besonders beachten:

- ✅ **Differenzierung** (qualitativ verschiedene Aufgaben für A/B-Gruppen)
- ✅ **Binnendifferenzierung** (Peer-Unterstützung, gemischte Tandems)
- ✅ **Sprechaktivität** (alle SuS sollen mehrfach sprechen)
- ✅ **Kommunikative Kompetenz** (echte Interaktion, nicht nur Reproduktion)
- ✅ **Scaffolding** (gestufte Hilfen für schwächere SuS)
- ✅ **Lernziel-Transparenz** (klare Ziele, die SuS verstehen)

## TEIL 8: JAVASCRIPT FÜR LERNPFADE

### 8.1 Eingabetoleranz für Spanisch

Schüler verwenden deutsche Tastaturen. Akzeptiere Eingaben auch ohne:
- Akzente (á, é, í, ó, ú → a, e, i, o, u)
- Tilde (ñ → n)
- Umgekehrte Satzzeichen (¿, ¡)
- Groß-/Kleinschreibung

```javascript
const SpanishTolerance = {
    normalize: function(text) {
        if (!text) return '';
        return text
            .toLowerCase()
            .trim()
            .replace(/^[¿¡]+/, '')
            .replace(/[?!.]+$/, '')
            .normalize('NFD')
            .replace(/[\u0300-\u036f]/g, '')
            .replace(/\s+/g, ' ')
            .trim();
    },

    match: function(userInput, correct) {
        return this.normalize(userInput) === this.normalize(correct);
    }
};

// Shorthand
function spanishMatch(input, correct) {
    return SpanishTolerance.match(input, correct);
}
```
