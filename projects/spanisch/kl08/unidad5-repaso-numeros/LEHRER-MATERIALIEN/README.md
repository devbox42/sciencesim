### Workflow Options

**A) Teacher-led (recommended)**
1. PPTX on projector
2. Book S. 78-85 for input
3. AB tasks after each input phase
4. Solutions via PPTX
5. Optional: QR-TEST as exit ticket

**B) Self-paced / Hybrid**
1. Distribute AB + QR-LERNPFAD
2. Students work through Lernpfad
3. Lernpfad guides AB completion
4. Teacher available for support
5. Optional: QR-TEST for assessment

**C) Homework / Review**
1. Lesson done previously
2. AB as revision homework
3. QR-LERNPFAD for self-check
4. QR-TEST as formative check

**D) Offline / No devices**
1. Print LB + AB for each student
2. LB provides all input (vocabulary, grammar)
3. Students read LB, then complete AB tasks
4. LB has "→ AB Aufgabe X" references
5. Check with ML (teacher) or PPTX

---

### Assessment with PLATZKARTEN

1. Hand out one card per student
2. Student scans QR → takes test
3. When done: student shows result
4. Teacher notes points/grade on card
5. Collect card immediately
6. Keep cards as paper record
