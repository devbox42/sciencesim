# Unidad 5 - Repaso: La ropa y los adjetivos

**Class 8 | 45 min | Paper-based**

## Materials

| File | Description |
|------|-------------|
| `LB-05-repaso-ropa.pdf` | Learning sheet with vocabulary, grammar rules, and phrases |
| `AB-05-repaso-ropa.pdf` | Worksheet with exercises (25 pts + extension) |
| `ML-05-repaso-ropa.pdf` | Answer key (solutions in red) |

## Usage

1. Distribute **LB** for vocabulary review (chorally)
2. Distribute **AB** for individual practice
3. Use **ML** for correction / self-check

## Content

- 12 clothing items (la camiseta, el jersey, los pantalones, etc.)
- 10 adjectives (bonito, moderno, comodo, elegante, largo, corto, etc.)
- Adjective agreement: -o/-a vs. -e
- Speaking phrases: "El/La ... es muy ...", "Me gusta ... porque ..."
