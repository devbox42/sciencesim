## Fachliche Tiefe

| Prüfpunkt | Antwort |
|-----------|---------|
| **Warum-Kette** | Wertigkeiten geben an, wie viele Bindungen ein Atom eingehen KANN – dies bestimmt das Verhältnis in Verbindungen. Beim Ausgleich sorgen wir dafür, dass Atome erhalten bleiben. |
| **Vorwissen-Brücke** | "Händezahl": Mg hat 2 "Hände", O hat 2 "Hände" → 1:1 passt. Al hat 3 "Hände", O hat 2 → braucht kgV (6) → Al₂O₃ |
| **Warum nicht Elektronenkonfiguration?** | Ionenbindung kommt erst Kl. 9. Jetzt pragmatisch: Wertigkeiten als Tabellenwerte lernen, Oktettregel erklärt nicht Fe(II/III). |
| **Häufiger Denkfehler** | Index und Vorfaktor verwechseln: "2 MgO" ≠ "MgO₂"! Index ändert Stoffart, Vorfaktor nur Menge. |
| **Häufiger Denkfehler 2** | O₂ vergessen: Sauerstoff liegt als O₂ vor, nicht als einzelnes O-Atom. |

## Motivation & Alltagsrelevanz (KRITISCH!)

**Einstiegsfrage:** "Warum explodiert ein Airbag in genau 30 Millisekunden – nicht langsamer, nicht schneller?"

**Antwort:** Weil Ingenieure die EXAKTEN Mengen berechnet haben!
- Airbag: 2 NaN₃ → 2 Na + 3 N₂ (Natriumazid zerfällt → Stickstoff bläst Airbag auf)
- Falsche Menge = zu wenig Gas (Airbag schlaff) oder zu viel (Airbag platzt)

**Weitere Beispiele im LP:**
| Anwendung | Reaktion | Warum Mengen wichtig? |
|-----------|----------|----------------------|
| **Airbag** | Natriumazid → Stickstoff | Exakt 67 Liter Gas in 30 ms |
| **Thermit-Schweißen** | 2 Al + Fe₂O₃ → Al₂O₃ + 2 Fe | Bahnschienen verschweißen |
| **Wunderkerze** | Fe + O₂ → Fe₃O₄ | Funkeneffekt durch Eisenspäne |
| **Rostschutz** | 4 Fe + 3 O₂ → 2 Fe₂O₃ | Wie viel Zink braucht die Beschichtung? |

**Kernbotschaft:** "Reaktionsgleichungen sind die REZEPTE der Chemie. Ohne genaue Mengen geht es schief!"

## Der 5-Schritte-Algorithmus (KERN DER STUNDE)

```
┌─────────────────────────────────────────────────────────────────┐
│  ALGORITHMUS: Reaktionsgleichung aufstellen und ausgleichen     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1️⃣ WORTGLEICHUNG                                               │
│     Metall + Sauerstoff → Metalloxid                            │
│     Beispiel: Aluminium + Sauerstoff → Aluminiumoxid            │
│                                                                 │
│  2️⃣ WERTIGKEITEN ABLESEN                                        │
│     Al = III, O = II                                            │
│                                                                 │
│  3️⃣ FORMEL MIT KREUZREGEL                                       │
│     Al(III) + O(II) → Al₂O₃                                     │
│     (III→3 wird Index von O, II→2 wird Index von Al)            │
│                                                                 │
│  4️⃣ GLEICHUNG AUFSTELLEN (noch unausgeglichen)                  │
│     Al + O₂ → Al₂O₃                                             │
│     Achtung: Sauerstoff ist O₂!                                 │
│                                                                 │
│  5️⃣ ATOME ZÄHLEN + AUSGLEICHEN                                  │
│     Links:  Al = 1, O = 2                                       │
│     Rechts: Al = 2, O = 3                                       │
│     → Nicht ausgeglichen! Vorfaktoren anpassen:                 │
│                                                                 │
│     kgV(2,3) für O = 6 → brauche 6 O auf beiden Seiten          │
│     Links: 3 O₂ = 6 O ✓                                         │
│     Rechts: 2 Al₂O₃ = 6 O ✓                                     │
│     → 2 Al₂O₃ braucht 4 Al                                      │
│                                                                 │
│  ✅ ERGEBNIS: 4 Al + 3 O₂ → 2 Al₂O₃                             │
│                                                                 │
│  🔍 KONTROLLE: Al: 4=4 ✓  O: 6=6 ✓                              │
└─────────────────────────────────────────────────────────────────┘
```

## Feedback-Design (elaboriert!)

**Im LP: Nicht nur "richtig/falsch", sondern prozessorientiert:**

| Fehlertyp | Feedback-Text |
|-----------|---------------|
| Index falsch | "Die Indices stimmen nicht. Hast du die Kreuzregel angewendet? Al(III) + O(II) → Index von Al wird 2, Index von O wird 3." |
| Vorfaktor vergessen | "Die Formel ist richtig, aber die Gleichung ist nicht ausgeglichen! Zähle: Links __ O, Rechts __ O. Wie viele O₂ brauchst du?" |
| O statt O₂ | "Achtung: Sauerstoff kommt in der Natur als O₂ vor, nicht als einzelnes O! Korrigiere die linke Seite." |
| Index/Vorfaktor verwechselt | "Du hast den Index geändert – aber das ändert den STOFF! MgO₂ ist ein anderer Stoff als MgO. Nutze nur Vorfaktoren VOR der Formel." |
| Richtig | "✓ Perfekt! Die Gleichung ist ausgeglichen: Links __ Atome, Rechts __ Atome." |

**Scaffolding-Stufen bei Fehlern:**
1. Hinweis: "Zähle nochmal die O-Atome."
2. Teillösung: "Links: 2 O. Rechts: 3 O. Das passt nicht."
3. Nächster Schritt: "Finde das kgV von 2 und 3. Wie viele O brauchst du auf beiden Seiten?"
