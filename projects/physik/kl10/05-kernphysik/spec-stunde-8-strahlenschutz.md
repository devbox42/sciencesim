## Fachlicher Hintergrund

### Dosisbegriffe

| Begriff | Symbol | Einheit | Bedeutung |
|---------|--------|---------|-----------|
| **Aktivität** | A | Becquerel (Bq) | Zerfälle pro Sekunde |
| **Energiedosis** | D | Gray (Gy) | Absorbierte Energie pro Masse (J/kg) |
| **Äquivalentdosis** | H | Sievert (Sv) | Biologische Wirksamkeit (= D × Qualitätsfaktor) |

**Qualitätsfaktoren:**
| Strahlung | Q |
|-----------|---|
| γ, β | 1 |
| α | 20 |
| Neutronen | 5-20 |

→ α-Strahlung ist bei gleicher Energiedosis 20× schädlicher!

### Strahlenbelastung

**Natürliche Strahlung (~2,1 mSv/Jahr in Deutschland):**
- Kosmische Strahlung: ~0,3 mSv
- Terrestrische Strahlung: ~0,4 mSv
- Radon (Einatmen): ~1,1 mSv
- Nahrung (K-40): ~0,3 mSv

**Künstliche Strahlung (~1,8 mSv/Jahr):**
- Medizin (Röntgen, CT): ~1,7 mSv
- Sonstiges: ~0,1 mSv

**Gesamt: ~4 mSv/Jahr** (Durchschnitt Deutschland)

**Grenzwerte:**
| Personengruppe | Grenzwert/Jahr |
|----------------|----------------|
| Bevölkerung | 1 mSv |
| Beruflich exponiert | 20 mSv |
| Notfallhelfer | 250 mSv (einmalig) |

### Die 3 A's des Strahlenschutzes

```
┌─────────────────────────────────────────────────────────────┐
│                    STRAHLENSCHUTZ                           │
│                                                             │
│    1. ABSTAND        2. ABSCHIRMUNG      3. AUFENTHALTSZEIT │
│                                                             │
│    Je größer,        Material zwischen   Je kürzer,         │
│    desto weniger     Quelle und Körper   desto weniger      │
│    Dosis             (Blei, Beton)       Dosis              │
│                                                             │
│    D ~ 1/r²          Je nach Strahlung   D ~ t              │
│                      (α: Papier,                            │
│                       β: Alu,                               │
│                       γ: Blei)                              │
└─────────────────────────────────────────────────────────────┘
```

### Biologische Wirkung

**Akute Strahlenschäden (hohe Dosis, kurze Zeit):**
| Dosis | Wirkung |
|-------|---------|
| <0,25 Sv | Keine merklichen Symptome |
| 0,25-1 Sv | Übelkeit, Blutbildveränderungen |
| 1-3 Sv | Strahlenkrankheit (Erbrechen, Haarausfall) |
| 3-5 Sv | 50% Sterblichkeit ohne Behandlung |
| >7 Sv | Tödlich |

**Späte Strahlenschäden (niedrige Dosis, lange Zeit):**
- Erhöhtes Krebsrisiko
- Genetische Schäden (Keimzellen)
- Kein Schwellenwert bekannt!

### Medizinische Anwendungen

| Anwendung | Strahlung | Zweck |
|-----------|-----------|-------|
| Röntgen | Röntgenstrahlen | Knochenbrüche, Zähne |
| CT | Röntgenstrahlen | 3D-Schnittbilder |
| Szintigraphie | γ (Tc-99m) | Organfunktion |
| PET | β⁺ (F-18) | Stoffwechsel, Tumore |
| Strahlentherapie | γ, Elektronen | Krebsbehandlung |
| Brachytherapie | γ (I-125) | Lokale Tumorbehandlung |

**Prinzip Strahlentherapie:**
- Tumorzellen teilen sich schnell → empfindlicher für Strahlung
- Gezielte Bestrahlung des Tumors
- Gesundes Gewebe wird geschont (Mehrfeldertechnik)
