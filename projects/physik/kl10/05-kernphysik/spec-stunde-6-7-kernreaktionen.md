## Fachlicher Hintergrund

### Massendefekt und Bindungsenergie

**Beobachtung:** Die Masse eines Kerns ist KLEINER als die Summe seiner Bestandteile!

```
Beispiel Helium-4:
2 Protonen:   2 × 1,0073 u = 2,0146 u
2 Neutronen:  2 × 1,0087 u = 2,0174 u
─────────────────────────────────────
Summe:                      4,0320 u
Tatsächliche Masse He-4:    4,0026 u
─────────────────────────────────────
Massendefekt Δm:            0,0294 u
```

**Erklärung:** Die "fehlende" Masse wurde in Bindungsenergie umgewandelt.

**Einstein'sche Formel:**
```
E = m · c²

mit c = 3 × 10⁸ m/s (Lichtgeschwindigkeit)
```

**Umrechnung:** 1 u ≈ 931 MeV (Megaelektronenvolt)

**Bindungsenergie pro Nukleon:**
- Leichte Kerne: niedrig
- Mittelschwere Kerne (Fe-56): Maximum
- Schwere Kerne: wieder niedriger

→ Energiegewinn möglich durch:
- **Spaltung** schwerer Kerne (U-235 → mittelschwer)
- **Fusion** leichter Kerne (H → He)

### Kernspaltung

**Entdeckung:** Otto Hahn, Lise Meitner, Fritz Straßmann (1938)

**Prozess:**
```
²³⁵₉₂U + ¹₀n → ²³⁶₉₂U* → ¹⁴¹₅₆Ba + ⁹²₃₆Kr + 3 ¹₀n + Energie

(oder andere Spaltprodukte – viele Möglichkeiten!)
```

**Energiefreisetzung:** ~200 MeV pro Spaltung

**Kettenreaktion:**
1. Neutron trifft U-235
2. Kern spaltet sich, setzt 2-3 Neutronen frei
3. Diese Neutronen spalten weitere Kerne
4. Exponentielles Wachstum!

**Kontrolle:**
- **Unkontrolliert:** Atombombe (kritische Masse überschritten)
- **Kontrolliert:** Kernreaktor (Moderator, Steuerstäbe)

### Kernkraftwerk – Aufbau

```
┌─────────────────────────────────────────────────────────────┐
│                    KERNKRAFTWERK (DWR)                      │
│                                                             │
│  ┌──────────┐      ┌──────────┐      ┌──────────┐          │
│  │ Reaktor  │──────│ Dampf-   │──────│ Generator│──→ Strom │
│  │ (Wärme)  │      │ erzeuger │      │ (Turbine)│          │
│  └──────────┘      └──────────┘      └──────────┘          │
│       │                                    │               │
│       │         Primärkreislauf            │               │
│       │         (radioaktiv!)              │               │
│       │                                    │               │
│       └────────────────────────────────────┘               │
│                                        Kühlturm            │
│                                        (Kondensator)       │
└─────────────────────────────────────────────────────────────┘
```

**Komponenten:**
| Teil | Funktion |
|------|----------|
| Brennstäbe | Enthalten U-235 (angereichertes Uran) |
| Moderator | Bremst Neutronen (Wasser, Graphit) |
| Steuerstäbe | Absorbieren Neutronen → Leistungsregelung |
| Kühlmittel | Transportiert Wärme (Wasser) |
| Containment | Sicherheitshülle |

### Kernfusion

**Prozess in der Sonne:**
```
4 ¹₁H → ⁴₂He + 2 e⁺ + 2 ν + Energie

(vereinfacht: Proton-Proton-Kette)
```

**Energiefreisetzung:** ~26 MeV (für 4 H → He)

**Voraussetzungen:**
- Extrem hohe Temperatur (>100 Mio. °C)
- Hoher Druck (Plasma)
- → Überwindung der elektrostatischen Abstoßung

**Fusionsreaktor (Forschung: ITER):**
- Deuterium + Tritium → Helium + Neutron + Energie
- Magnetischer Einschluss (Tokamak)
- Noch nicht energetisch rentabel (Forschung!)

### Vergleich Spaltung – Fusion

| Aspekt | Kernspaltung | Kernfusion |
|--------|--------------|------------|
| Prozess | Schwerer Kern → leichtere | Leichte Kerne → schwerer |
| Brennstoff | U-235, Pu-239 | Deuterium, Tritium |
| Energie/Masse | ~80 TJ/kg | ~340 TJ/kg |
| Abfall | Langlebig, hochradioaktiv | Kurzlebig, weniger |
| Risiko | GAU möglich | Kaum (Reaktion stoppt) |
| Stand | Technisch ausgereift | Forschungsstadium |
