## 4. Simulations-Spezifikationen

### ANI-01: Klotz anschieben (Haftreibung → Gleitreibung)

```
┌─────────────────────────────────────────┐
│  Kraftpfeil →→→→→→→→                    │
│  ┌───────┐                              │
│  │ KLOTZ │  ════════════════════════    │
│  └───────┘       Tisch                  │
│                                         │
│  Kraft: ▓▓▓▓▓▓▓░░░ 7 N                  │
│  Status: "Haftreibung - Klotz bewegt    │
│           sich noch nicht"              │
│                                         │
│  [▶ Play]  [↺ Reset]                    │
└─────────────────────────────────────────┘
```

**Verhalten:**
1. Kraft steigt langsam (0 → 10 N)
2. Bei ~8 N: Klotz beginnt plötzlich zu gleiten
3. Während Gleiten: Kraft fällt auf ~5 N (Gleitreibung < Haftreibung)
4. Textanzeige wechselt

### ANI-02: Auto bremst

```
┌─────────────────────────────────────────┐
│  🚗→→→→→→→→→→→→→→→ · · · · · STOP      │
│  ════════════════════════════════════   │
│                                         │
│  Geschwindigkeit: 50 km/h → 0 km/h      │
│  Bremsweg: 12 m                         │
│                                         │
│  Haftreibung Reifen-Straße:             │
│  ▓▓▓▓▓▓▓▓▓▓ (hoch = kurzer Bremsweg)    │
│                                         │
│  [▶ Play: Trockene Straße]              │
│  [▶ Play: Nasse Straße]                 │
└─────────────────────────────────────────┘
```

**Verhalten:**
- Trockene Straße: Bremsweg 12 m, Auto stoppt rechtzeitig
- Nasse Straße: Bremsweg 24 m, Auto rutscht über Markierung
- Zeigt: Weniger Reibung = längerer Bremsweg
