\begin{center}
{\LARGE\bfseries Das Apple-Ökosystem}\\[4pt]
{\large Wie Apple-ID, iCloud, App Store und Co. zusammenhängen}
\end{center}

\vspace{2mm}
\hrule
\vspace{4mm}

\begin{multicols}{2}

\section*{Die Grundidee}

\begin{center}
\fcolorbox{appleblue}{appleblue!8}{%
\begin{minipage}{0.9\linewidth}
\vspace{2mm}
\textbf{Deine Apple-ID ist der Schlüssel zu allem.}\\[4pt]
Alle Apple-Dienste sind mit deiner Apple-ID verbunden. Meldest du dich auf einem neuen Gerät an, hast du automatisch Zugriff auf alle deine Daten und Einkäufe.
\vspace{2mm}
\end{minipage}}
\end{center}

\vspace{2mm}

\textbf{Analogie: Wie ein Hotelgast}
\begin{itemize}
\item Deine \textbf{Apple-ID} = deine Gästekarte
\item Mit dieser Karte öffnest du: Zimmer (iCloud), Restaurant (App Store), Spa (Apple Music), Concierge (Support)
\item Egal welches Hotel der Kette (= welches Apple-Gerät) – deine Karte funktioniert überall
\end{itemize}

\vspace{4mm}

\section*{Die wichtigsten Dienste}

\begin{tabular}{@{}p{2.5cm}p{9cm}@{}}
\textbf{\textcolor{icloudblue}{iCloud}} &
Dein Speicher im Internet: Fotos, Kontakte, Kalender, Notizen, Backups. Synchronisiert automatisch zwischen allen Geräten. \\[4mm]

\textbf{\textcolor{appstoreblue}{App Store}} &
Hier kaufst/lädst du Apps. Einmal gekauft = auf allen deinen Geräten verfügbar. \\[4mm]

\textbf{\textcolor{findmygreen}{\glqq Wo ist?\grqq}} &
Findet deine verlorenen Geräte auf einer Karte. Kann Geräte sperren oder löschen. \\[4mm]

\textbf{\textcolor{imessagegreen}{iMessage}} &
Nachrichten zwischen Apple-Geräten (blau). Kostenlos, über Internet. \\[4mm]

\textbf{FaceTime} &
Video- und Telefonanrufe zwischen Apple-Geräten. Kostenlos. \\[4mm]

\textbf{\textcolor{photospurple}{Fotos}} &
Deine Bilder – mit iCloud auf allen Geräten synchron. \\
\end{tabular}

\columnbreak

\textbf{Das bedeutet:}
\begin{itemize}
\item Ein Foto auf dem iPhone → erscheint automatisch auf iPad und Mac
\item Eine App auf dem iPhone gekauft → kostenlos auf iPad laden
\item iPhone verloren → vom Mac aus orten und sperren
\end{itemize}

\end{multicols}

\vspace{2mm}
\end{minipage}}

\begin{center}
{\LARGE\bfseries Die Dienste im Detail}\\[4pt]
{\large Was macht was? Wo finde ich es?}
\end{center}

\vspace{2mm}
\hrule
\vspace{4mm}

% Vier Boxen
\begin{minipage}[t]{0.48\linewidth}

% iCloud Box
\fcolorbox{icloudblue}{icloudblue!8}{%
\begin{minipage}{\linewidth}
\vspace{3mm}
\begin{center}
{\large\bfseries\textcolor{icloudblue}{iCloud -- Dein Speicher in der \glqq Wolke\grqq{}}}
\end{center}
\vspace{2mm}

\textbf{Was wird gespeichert?}
\begin{itemize}
\item Fotos und Videos
\item Kontakte und Kalender
\item Notizen und Erinnerungen
\item Backups deines iPhones
\item Dokumente (Pages, Numbers...)
\item Passwörter (Schlüsselbund)
\end{itemize}

\vspace{2mm}
\textbf{Wo finde ich es?}
\begin{itemize}
\item iPhone/iPad: Einstellungen → [Dein Name] → iCloud
\item Mac: Systemeinstellungen → Apple-ID → iCloud
\item Im Browser: \textbf{icloud.com}
\end{itemize}

\vspace{2mm}
\textbf{Speicherplatz:}\\
5 GB kostenlos. Mehr kostet: 50 GB = 0,99€/Monat

\vspace{2mm}
\end{minipage}}

\vspace{4mm}

% Wo ist? Box
\fcolorbox{findmygreen}{findmygreen!8}{%
\begin{minipage}{\linewidth}
\vspace{3mm}
\begin{center}
{\large\bfseries\textcolor{findmygreen}{\glqq Wo ist?\grqq{} – Geräte finden}}
\end{center}
\vspace{2mm}

\textbf{Was kann es?}
\begin{itemize}
\item Zeigt alle deine Geräte auf einer Karte
\item Lässt ein verlorenes Gerät klingeln
\item Sperrt ein gestohlenes Gerät aus der Ferne
\item Löscht alle Daten aus der Ferne (Notfall!)
\end{itemize}

\vspace{2mm}
\textbf{Wo finde ich es?}
\begin{itemize}
\item App \glqq Wo ist?\grqq{} auf iPhone/iPad/Mac
\item Im Browser: \textbf{icloud.com/find}
\end{itemize}

\vspace{2mm}
\textbf{Wichtig:} Funktioniert nur, wenn das Gerät online ist (WLAN oder Mobilfunk).

\vspace{2mm}
\end{minipage}}

\end{minipage}%
\hfill%
\begin{minipage}[t]{0.48\linewidth}

% App Store Box
\fcolorbox{appstoreblue}{appstoreblue!8}{%
\begin{minipage}{\linewidth}
\vspace{3mm}
\begin{center}
{\large\bfseries\textcolor{appstoreblue}{App Store – Apps laden}}
\end{center}
\vspace{2mm}

\textbf{Was ist das?}
\begin{itemize}
\item Der \glqq Laden\grqq{} für Apps auf deinem Gerät
\item Kostenlose und kostenpflichtige Apps
\item Einmal gekauft = auf allen Geräten nutzbar
\end{itemize}

\vspace{2mm}
\textbf{Wo finde ich es?}
\begin{itemize}
\item App \glqq App Store\grqq{} auf iPhone/iPad/Mac
\end{itemize}

\vspace{2mm}
\textbf{Bezahlung:}
\begin{itemize}
\item Hinterlegt in deiner Apple-ID
\item Kreditkarte, PayPal oder Guthabenkarte
\item Einstellungen → [Dein Name] → Zahlung \& Versand
\end{itemize}

\vspace{2mm}
\textbf{Käufe wiederherstellen:}\\
Gekaufte Apps → App Store → [Dein Bild] → Käufe

\vspace{2mm}
\end{minipage}}

\vspace{4mm}

% iMessage/FaceTime Box
\fcolorbox{imessagegreen}{imessagegreen!8}{%
\begin{minipage}{\linewidth}
\vspace{3mm}
\begin{center}
{\large\bfseries\textcolor{imessagegreen}{iMessage \& FaceTime}}
\end{center}
\vspace{2mm}

\textbf{iMessage:}
\begin{itemize}
\item Nachrichten zwischen Apple-Geräten
\item \textbf{Blaue Blasen} = iMessage (kostenlos, über Internet)
\item Grüne Blasen = normale SMS (kann kosten)
\end{itemize}

\vspace{2mm}
\textbf{FaceTime:}
\begin{itemize}
\item Videoanrufe zwischen Apple-Geräten
\item Kostenlos über WLAN oder mobile Daten
\item Auch reine Sprachanrufe möglich
\end{itemize}

\vspace{2mm}
\textbf{Wo aktivieren?}\\
Einstellungen → Nachrichten / FaceTime

\vspace{2mm}
\end{minipage}}

\end{minipage}
