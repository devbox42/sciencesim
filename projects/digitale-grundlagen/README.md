# Digitale Grundlagen

## Zielgruppe

**Digitale "Analphabeten" 65+** – Menschen, die die technologische Entwicklung seit ca. 1990 weitgehend verpasst haben und nun "überlebensfähig" in der digitalen Welt werden müssen.

## Konzept: Die digitale Fahrschule

Wie eine Fahrschule: Nicht alles verstehen, aber sicher ankommen.

| Fahrschule | Digitale Grundlagen |
|------------|---------------------|
| Verkehrsregeln | Passwörter, Sicherheit, Datenschutz |
| Fahrzeugbedienung | Smartphone/Tablet/PC bedienen |
| Gefahrenerkennung | Phishing, Betrug, Spam erkennen |
| Erste Hilfe | Was tun bei Problemen, an wen wenden |
| Prüfung | Selbst-Checkliste "Ich kann das" |

## Didaktische Prinzipien

1. **Alltagsanalogien statt Technik-Jargon**
   - SIM-PIN = Briefkastenschlüssel
   - Geräte-Code = Hausschlüssel
   - Apple-ID = Ausweis bei einer Firma

2. **Überleben statt Meistern**
   - Was MUSS ich wissen?
   - Was passiert, wenn ich X falsch mache?
   - An wen wende ich mich bei Problemen?

3. **Visuell und haptisch**
   - Große Schrift (11pt+)
   - Farbcodierung
   - Checklisten zum Abhaken
   - Platz für eigene Notizen

4. **Sicherheit betonen**
   - Konsequenzen klar benennen
   - "WARNUNG"-Boxen für kritische Punkte
   - Vertrauenspersonen einbeziehen

## Format

Jede Einheit besteht aus:
- **Infoblatt** (1-2 Seiten, große Schrift)
- **Übungen** (Zuordnung, Richtig/Falsch, Situationen)
- **Checkliste** ("Das habe ich verstanden")
- **Notfall-Notizen** (Zum Ausfüllen und Aufbewahren)
